Títulos

Podemos utilizar as tags <h1> até <h6> para "dizer" para o html que queremos escrever um título. E os números na frente da letra "h" significam os níveis de títutos que queremos, abaixo estão os exemplos:

  1. Nível 1

  2. Nível 2

  3. Nível 3

  4. Nível 4

  5. Nível 5
  6. Nível 6

Parágrafos

A tag <p> é utilizada para escrever os parágrafos do site e dentro dela podemos utilizar diversas tags de formatação de texto, como os exemplo: Negrito, Itálico, Sublinhado, Marcação, Textos inferiores e Textos superiores

Podemos utilizar citações também com a tag <blockquote> como esta, o texto fica com uma margem maior que os outros.
